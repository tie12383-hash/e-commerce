# E-commerce

Cистема управления товарами и категориями

## Функциональность

### Текущая версия:

- **Класс Product**: представление товара с атрибутами:
  - Название (name)
  - Описание (description)
  - Цена (price)
  - Количество на складе (quantity)

- **Класс Category**: представление категории товаров с атрибутами:
  - Название (name)
  - Описание (description)
  - Список товаров (products)
  - Автоматический подсчет категорий (category_count)
  - Автоматический подсчет товаров (product_count)

- **Загрузка данных**: функция для импорта категорий и товаров из JSON-файла

### Требования

- Python 3.8+
- Poetry для управления зависимостями

### Установка и запуск

```bash
# Установка зависимостей
poetry install

# Запуск тестов
