# E-commerce Core - Версия 2.0

## Новая функциональность (Защита данных)

### Приватные атрибуты

1. **Класс `Category`:**
   - Атрибут `products` стал приватным (`__products`)
   - Метод `add_product()` для безопасного добавления товаров
   - Геттер через `@property` для получения форматированного списка товаров

2. **Класс `Product`:**
   - Атрибут `price` стал приватным (`_price`)
   - Реализованы геттер и сеттер с проверками:
     - Цена не может быть нулевой или отрицательной
     - При понижении цены требуется подтверждение пользователя

### Класс-методы

**Класс `Product`:** 
- Добавлен класс-метод `new_product()`
- Автоматически обрабатывает дубликаты:
  - Складывает количество одинаковых товаров
  - Выбирает максимальную цену при конфликте
