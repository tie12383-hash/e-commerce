# E-commerce Core - Версия 2.0

## Реализованная функциональность

### Абстрактные классы
- Добавлен абстрактный класс `BaseProduct` - родитель для всех продуктов
- Добавлен абстрактный класс `BaseContainer` - родитель для `Category` и `Order`
- Реализованы обязательные методы в абстрактных классах

### Класс-миксин
- Реализован миксин `ReprMixin` для логирования создания объектов
- Миксин добавляет метод `__repr__` для строкового представления объектов
- Миксин выводит в консоль информацию при создании объектов

### Новый класс Order
- Реализован класс `Order` для работы с заказами
- Заказ содержит один товар и количество
- Реализована проверка доступного количества товара
- Добавлен метод обработки заказа

### Тестирование
- Написаны новые тесты для абстрактных классов и миксина
- Покрытие тестами > 75%
- Все существующие тесты проходят без ошибок
