# E-commerce Core - Версия 2.0

## Реализованная функциональность

### Наследование классов
1. **Базовый класс Product** - основной класс для всех товаров
2. **Класс Smartphone** - для смартфонов с дополнительными атрибутами:
   - Производительность (efficiency)
   - Модель (model)
   - Объем встроенной памяти (memory)
   - Цвет (color)
3. **Класс LawnGrass** - для газонной травы с дополнительными атрибутами:
   - Страна-производитель (country)
   - Срок прорастания (germination_period)
   - Цвет (color)

### Ограничения на операции
1. **Сложение продуктов**:
   - Можно складывать только объекты одинакового класса
   - Попытка сложить объекты разных классов вызывает `TypeError`
   - Используется проверка `type(self) == type(other)`

2. **Добавление в категорию**:
   - Можно добавлять только объекты класса `Product` или его наследников
   - Используется проверка `isinstance()` и `issubclass()`
   - Попытка добавить объект другого типа вызывает `TypeError`
